selector

selector의 경우 #의 id, .의 class, p의 태그 순으로 우선된다 그 이유는 id의 경우 유일무이 하고 class의 경우에는 일부분만 지정이 되기 때문이다. 태그는 그냥 포함이 되는 모든 부분을 말하기에 위로부터 내려오면서 순서가 정해진다. css 를 사용할 때는 style 태그를 사용한다 형태는 아래와 같다. tag { } 태두리는 border, 색은 color, 글자는 font나 text를 사용한다 display를 이용해서 grid를 사용할 수도 있고 div 는 block, span은 inline 을 나타내는 태그로 사용되기도 한다. mediaquery는 창의 크기를 조절하면서 나타내는 모습을 다르게 할 수도 있다
